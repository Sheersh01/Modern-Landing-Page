# Modern Landing Page

A visually stunning and interactive landing page built with HTML, CSS, and modern JavaScript animation libraries.

## 🚀 Technologies Used
- **HTML5 & CSS3**: For structure and styling.
- **JavaScript**: For logic and interaction.
- **GSAP (GreenSock Animation Platform)**: For high-performance, complex animations.
- **ScrollTrigger**: For scroll-based animations (GSAP plugin).
- **Three.js**: For 3D elements and WebGL rendering.
- **Shery.js**: For creating engaging hover effects, image distortions, and other visual interactions.

## 🌟 Features
- Dynamic text animations and transitions.
- Interactive elements and background effects.
- Smooth scroll-triggered animations.
- Responsive navigation bar with hover effects.

## 🛠️ Getting Started
To view the project locally:
1. Clone this repository or download the ZIP file.
2. Open the `index.html` file in your preferred web browser.
3. No build tools or package managers are required!
